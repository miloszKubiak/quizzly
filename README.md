## Quizzly

[![quizzly.jpg](https://i.postimg.cc/HnHmDHXK/quizzly.jpg)](https://postimg.cc/R6PyKkPQ)

Quiz application written in React using some of the most important functionalities, e.g. :
- React Hooks(useState, custom hooks)
- Context API
- Styled Components

Setup:
- install ```npm install```
- run ```npm start```
